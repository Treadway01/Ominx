## 🚀 Overview

Ominx is an AI-first web browser designed to make browsing smarter and more productive.

Instead of switching between your browser and AI tools, Ominx brings them together into one seamless experience. Search the web, visit websites, summarize pages, ask questions, and get instant AI-powered assistance—all from one place.

Whether you're a student, developer, researcher, or everyday internet user, Ominx helps you browse faster and think smarter.

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/ominx.git
```

Navigate into the project

```bash
cd ominx
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
